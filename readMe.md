# Калькулятор координат фаски для ЧПУ

https://popsalupa.github.io/bigWork/

Веб-приложение для расчёта параметров фаски при программировании токарной обработки на станках с ЧПУ.

## Возможности

- Два режима расчёта:
  1. По начальному диаметру, углу и глубине
  2. По начальному и конечному диаметрам, углу
- Визуализация фаски с размерами
- Генерация примера G-кода (Fanuc)
- Автоматический пересчёт при изменении параметров

## Использование

1. Откройте `index.html` в браузере
2. Выберите режим расчёта
3. Введите параметры фаски
4. Получите координаты для программирования

## Файлы

- `index.html` - основная HTML структура
- `style.css` - стили оформления
- `script.js` - логика расчётов и визуализации

## Технологии

- HTML5
- CSS3
- JavaScript (ES6+)
- Canvas API для визуализации

## Поддерживаемые браузеры

Все современные браузеры (Chrome, Firefox, Safari, Edge)
